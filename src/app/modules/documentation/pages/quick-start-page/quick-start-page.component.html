Requirements:

  1. Mac OS X, Windows, or Linux
  2. Node.js v6.5 or newer
  3. Angular 9

Quick Start:

  1. Run npm install
  2. Run npm start

There are also other npm tasks:

  1. npm build : if you need just to build the app
  2. npm lint : to check the source code for syntax errors and potential issues

For more instruction please refer to Angular Material Admin Full readme.md.
